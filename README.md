## Kullanım
Çalışmaları yerel makinenizde derlemek için:
1. İlgili haftanın dizinine gidin.
2. `make` komutunu çalıştırın.
3. Oluşan çalıştırılabilir dosyayı `./dosya_adi` şeklinde çalıştırın.

## Notlar
* Çalışmalar, **Advanced Programming in the UNIX Environment (APUE)** standartlarına uygun olarak geliştirilmiştir.
* Geliştirme ortamı: UNIX/Linux (ssh erişimli sunucu).
